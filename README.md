# Crafting Curves in SwiftUI

> An interactive visual blog post by [Kashif Hussain](https://in.linkedin.com/in/kashif-hussain-ba5b0a197)

**Live site →** `https://hkashif722.github.io/swiftui-curves-blog/`

A hands-on guide to `addQuadCurve` in SwiftUI — covering concave panels, control point mechanics, and multi-peak waves. All concepts are interactive: drag the control points directly in the browser.

## What's inside

| Concept | Demo |
|---|---|
| Single concave panel | Drag yellow dot to feel the dip |
| Control point x & y | Independent sliders for each axis |
| Chained wave | 4 draggable control points |

## Local dev

```bash
npm install
npm run dev
```

## Deploy

Push to `main` — GitHub Actions builds and deploys automatically to GitHub Pages.
